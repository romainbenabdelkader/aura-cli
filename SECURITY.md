# Security Policy

aura-cli is a non-production demonstrator. It is not intended to secure real
assets or identities.

If you find a security issue — particularly in the cryptographic usage
(SHA3-256 hashing, Ed25519 key handling, signing/verification) — please report
it privately to the maintainer rather than opening a public issue.

Never commit or share private keys. The local TPKR registry and generated keys
are for demonstration only.
